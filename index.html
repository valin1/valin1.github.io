Hi, I'm Vanessa

I am a senior at UC Berkeley studying Computer Science and Applied Mathematics. Currently, I work as an undergraduate researcher in Berkeley Real-Time Intelligent Secure Explainable Systems Lab (RISELab). I am designing and building a high-level API for users to process and query video data, such that the architecture is optimized for multi-video applications with Maureen Daum, Brandon Haynes, and Professor Alvin Cheung. I am also studying variational autoencoders with an emphasis of physical applications with post-doctoral fellow N. Benjamin Erichson and Professor Michael Mahoney. This summer, I interned at Google as a software engineer intern on the PGP Search Ads Team.

Previously, I developed a suggestions system to promote better sustainable transportation habits on a platform, E-mission, with K. Shankari PhD, Professor David Culler, and Professor Randy Katz. Last summer at Google, I worked on building a hotel cancellation API to facilitate the direct cancellation process of a reservation with Google Hotels. Before that, I worked as a software developer intern at Sandia National Laboratories. There, I built an interactive data collection web app for the game SIGNAL.

projects

e-mission

A suggestions based system built on top of the pre-existing platform to encourage users to take more sustainable forms of transportations in their future trips.

Book on Google

Built a cancellation feature in which users will be able to use for Book on Google, a platform that facilitates hotel booking on Google without breaking the search flow.

omnitraffic

A web application that visualizes data analytics using the WebGl-powered framework, deck.gl to suggest alternative times based on traffic findings for city planners to potentially use and change current bus schedules.

PoNG

Project on Nuclear Gaming

Developed an interactive data collection web application for the game SIGNAL. The application allows proctors to input data collected from board game rounds and facilitate analysis of nuclear deterrence and conflict escalation.

face morph

Implemented a warping algorithm to create a "morph" animation of my face into another person's face. The morph is a simultaneous warp of the image shape and cross-dissolve of the image colors.

facial keypoint detection

Built convolutional neural networks (CNNs) to detect full facial keypoints, like eyes, brows, nose, lips, and chin, through PyTorch.

research

Forecasting Sequential Data using Consistent Koopman Autoencoders

Omri Azencot*, N. Benjamin Erichson*, Vanessa Lin, Michael W. Mahoney

[paper] Accepted to ICML 2020

In this work, we propose a novel Consistent Koopman Autoencoder model which, unlike the majority of existing work, leverages the forward and backward dynamics. The key to our approach is a new analysis that unravels the interplay between consistent dynamics and their associated Koopman operators.
